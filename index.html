What is Python?

Python is a powerful, high-level programming language that is widely used in web development, data science, automation, and more. Known for its readability and simplicity, Python is an excellent choice for beginners. This guide will introduce you to the basics of Python and help you get started on your coding journey.

Variables

Variables are used to store information that can be used and manipulated in your program.

name = "Alice" age = 25
Data Types

Python has several built-in data types:

  • Strings: Text data, enclosed in quotes.
    • greeting = "Hello, World!"
  • Integers: Whole numbers.
    • count = 10
  • Floats: Decimal numbers.
    • price = 19.99
  • Booleans: True or False values.
    • is_active = True
  • Sets: An unordered collection of data types.
    • set1 = set("freeCodeCamp") print("\nSet with the use of String: ") print(set1)
Lists

A list is an ordered collection of items.

mylist = ["apple", "banana", "cherry"]
Dictionaries

A dictionary stores data in key-value pairs.

person = {"name": "Alice", "age": 25}
IF Statements & Loops

IF Statements

Use if statements to execute code based on conditions.

age = 18 if age >= 18: print("You are an adult.") else: print("You are a minor.")

Loops

Loops allow you to execute code multiple times.

  • For Loop

    • For loops are used to iterate over a sequenece and execute a block of code for each element in the sequence.

    for i in range(5): print(i)

  • While Loop

    • A while loop repeatedly executes a block of code as long as a given condition is true.

    count = 0 while count < 5: print (count) count += 1
Functions

Functions are reusable blocks of code.

Defining a Function

def greet(name): return f"Hello, {name}!"

Calling a Function

message = greet("Alice") print(message)
